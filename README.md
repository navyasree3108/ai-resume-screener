
🤖 AI Resume Screening Web Application

📌 Overview
This project is an AI-powered resume screening system that matches candidate resumes with job descriptions using NLP techniques.

🚀 Features
- Compare resumes with job description
- Generate match percentage
- Rank candidates based on relevance
- Simple web interface using Flask

🧠 Technologies Used
- Python
- Flask
- NLP (TF-IDF, Cosine Similarity)
- Scikit-learn

▶️ How it Works
1. Enter job description
2. Enter candidate resumes
3. System analyzes and gives match scores

🌐 Deployment
Deployed using ngrok for public access.

- Add more candidates
- Improve UI design
- Use advanced ML models
